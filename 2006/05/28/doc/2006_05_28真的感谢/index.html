真的感谢

2006-05-28 02:35:12

就要高考了,回想高中的这三年真的没有什么留恋的说好的卢中是我的跳板过了这里我就离开也不想有什么留恋,很开心的也很无聊的过了两年,我真的没想过我会遇见你,也是说好的我要和你做最好的朋友心里确实是这么想的可你知道我为什么后来说爱你吗?其实那真的不是我的心里话我虽然喜欢你但我明白爱与喜欢的不同,我明白的很,在我们见面的那一刻我就知道我们之间是连朋友都做不了的,我明白我也不想说我爱你我只是喜欢你喜欢是发自内心的真的喜欢我曾想过如果我们只是很正常的认识的话我们会成为最好的朋友真的最好的朋友可都是我的卤莽我真的很后悔用那种方式和你认识可我没有任何办法我只能让它继续下去,我真的没有任何办法去挽回还好只剩下10天就要高考了高考就是分开了以后也没有机会在见面了,我曾说过我要去欧美可那都是假的我为什么要去因为你吗?如果你是我最爱的人我会毫不犹豫的去,可你不是,真的不是.我只想我们是好朋友是那种无话不说的好朋友你有烦恼的时候会想到我,可以找我聊天和我说话,但我知道那是不可能的,我不是傻子,我明白你的意思在你见到我的那一刻,我就知道我错了我应该把他埋在心底的高考之后就会忘记了我不懂男女之间的事情,我处理不好那些事情,我也不想去处理,就像你说的"我不明白什么是爱,我也不想去碰触."什么是爱?其实在我的心里早就冷了,我也是不想去接触,我不爱你但是我喜欢你,我说我爱你那全是假的,我只是喜欢你这一点你不用怀疑,这就是我说我欠你的一份情的原因,我说了"我爱你",我应该对我的话负责的,可是我却没有履行爱你的责任,因为我无法去对一个我不爱的人去表达我的爱意,我不是那种人,表面说一套背后做一套,我不想伤害你,所以我只能选择沉默和冷漠.对你也对我,说起来有点尴尬,还记得我们见面以后我在楼梯口碰见你的时候吗?其实我当时真的没看见你,我的眼睛坏了,正好那天我还没带眼镜,后来我想起的时候我感觉好象碰见你着,然后在那以后我发现以后我们见面的时候连句话都不说,我感觉就是因为那次我的印象在你的心里留下了不好的印象,我也只能编个谎言说我无法面对过去,不能说是谎言那是真的真的不赶面对过去,但是我发现我认识你以后我的心里发生了许多细小的变化,我开始审视我的过去,说好听点就是走出了一个人的阴影说不好听点就是我对女孩的态度改变了,不是那种痴情了,就是忘记了我现在都想不起来她的样子了,我还说过要等十年,呵~~现在想起来真的很可笑,时间真的可以冲淡一切的,以前不信着但是现在信了真的信了,所以说以后对你我也会…..漫漫的忘记的只要我们不见面,但我真的很开心能够能够和你成为朋友,我不希望我忘记你我希望我们以后可以常常联系,经常讲讲我们各自的生活,那样不是很好吗?最好的哥们儿呵呵是我违背了我最初的诺言我不想去澄清我们之间的误会了误会就误会吧.要不又能怎样呢?还有10天就考试了到时候就分开了在大的误会也会冲淡的时间是可以冲淡一切的呵呵!只是没能和你成为好哥们儿一直是我心里的遗憾.在这里也只能默默的祝福你了.找一个爱你的老公.其实还是有真爱的,它会是很平常很平常的但在你需要的时候它不会离开一直伴随在你的身旁.我祝福你!忠心的祝福!
